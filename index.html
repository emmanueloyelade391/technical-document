Introduction to HTML and CSS

This section is an introduction to HTML and CSS as a programming labguage.

HTML is a progamming language that is used to develop functional websites and customize webpages as users deem fit.

The other main component of HTML is CSS. CSS is the part of the language that allows users to decorate their websites. They can change colors, modify the sizes of elements, and etc.

HTML Elements

Elements are the pieces of code that give life to webpages.

Elements are the visible parts of code that you see on a web site. They are the called the "life" of a webpage bceause they give a webpage all of its functions.

There are a variety of elements that one can use to manufacture their website. These are some common elements:

  • Image element: img src= "link of picture"
  • DIV element: div class="name of class"
  • Anchor element:a href="hyperlink of other website/webpage"
  • Heading element:h1 "name of heading"
  • Doctype element:DOCTYPE "type of file the webpage is"
Example

Before you add the main content of your webpage, you need to lay the foundation of your webpage by setting it's language, it's size, characters, and linking it to CSS. You do that by writing:

  • !DOCTYPE
  • html lang="en"
  • head
  • meta charset="utf-8"
  • meta name="viewport" content="width=device-width, initial-scale=1.0"
  • link href="styles.css" rel="stylesheet"
  • head
  • html
CSS

Now that you have written the foundation of your webpage, you can customize your text by appyling CSS. When you used the link element you should have saw a box next to index.html that says style.css. Click that box and write:

  • .p {
  • font-style: arial;
  • font-color: grey;
  • }

This code will change the font of your text.

Putting It All Together

At this point of this short tutorial, you have established the pillars of your website. you can finally use the elements of HTML and the elements of CSS to build and change your website however you like. You should think of making a website like a journey because it sort of is. As you code you will go back to change aspects of your websites to fulfill new tasks. You will also make many mistakes. However, don't be discouraged! Part of being a programmer is making mistakes and learning from them.

Here are some tips for beginners who want to improve their coding skills quickly:

  • Collaborate with coders with similar experience so that you can get inspiration.

  • Have expert programmers debug your code and advise you during the web building process.

  • Go to programming websites to find code that can help your websites stand out! Here are some sites: freecodecamp, w3schools, and codeacademy.
